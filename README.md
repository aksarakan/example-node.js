# Nodejs Example

## Requirement
- nodejs >= 12 (tested)
- Composer

## Installation
- Clone or download this repository
- Install Dependency
- Change variable accordingly
- Run example

### Install Dependency
npm
```
npm i
```
yarn
```
yarn insall
```
### Change variable accordingly
change variable {path}, {token}, {type} inside example.js accordingly

### Running example
run following command
```
node example.js
```
